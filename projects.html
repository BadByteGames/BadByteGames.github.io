Glass Wall

The Glass Wall Game engine is the currently active project as there is only one dev team (me). The engine is designed to favor performance over how things look. The official repo is the technical up to date branch although most active development is done on my own fork here. Note: My fork is not guarenteed to be working in any way and should only be used to monitor my progress in development. The active goal right now is to make a 1.0 release.

Dragonhunt

The Dragonhunt Text Adventure Engine is an xml based text adventure engine. Currently, development is limited to bug patches until a dev team is found to work on it although you are free to add your own features and merge with the main repository. There is no official documentation available for it, and I have no good way of writing it, but if people ask for some docs, I will make it. There also used to be support for a web api but that broke and could be implemented by somebody. The project was made prior to when I knew how to write CMake code for projects.
