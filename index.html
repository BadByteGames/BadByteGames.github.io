 Byte Games is an awesome game company that makes fun games and tech demos. Check us out on github to see up-to-date code for projects and games. On this site you will find summaries of projects as well as contribution info and statuses regarding them. You can also check out the About Me section to view info about me (Duh) as developer.The projects list is shown under Projects. The Glass Wall Game Engine has its own seperate tab for info on it.
 Currently, I am working on The Glass Wall Game Engine (Glass Wall). Glass Wall is designed in C++ and is meant to focus on performance over graphics. This is to allow for complex simulation type games where you can have hundreds of moving parts. A level editor for it will be coming soon.
